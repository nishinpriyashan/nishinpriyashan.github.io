# Nishin Priyashan Portfolio

This repository contains a personal portfolio website for Nishin Priyashan.

## Files

- `nishin_portfolio (1).html` - Main portfolio HTML file.
- `assest/nishin.jpeg` - Profile photo used in the hero section.

## Notes

- Open `nishin_portfolio (1).html` in a web browser to view the portfolio.
- The hero photo source is set to `assest/nishin.jpeg`.

## Usage

1. Open the project folder in your preferred editor.
2. Double-click `nishin_portfolio (1).html` or open it in a browser.

## Recommended improvements

- Rename the `assest` folder to `assets` for standard spelling.
- Move CSS into a separate stylesheet for easier maintenance.
- Add more project pages or contact form functionality.
